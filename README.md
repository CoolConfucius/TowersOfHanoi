# Towers of Hanoi

A browser-based Towers of Hanoi puzzle with four difficulty levels. Move all disks from tower A to tower C — without ever placing a larger disk on top of a smaller one.

**[Live Demo](https://coolconfucius.github.io/TowersOfHanoi/)**

## How to Play

1. Select a difficulty level to start
2. Click a tower to pick up its top disk
3. Click another tower to place it (placing a larger disk on a smaller one is not allowed)
4. Move all disks to tower C to win

## Difficulty Levels

| Level  | Disks | Minimum Moves |
|--------|-------|---------------|
| Casual | 3     | 7             |
| Normal | 4     | 15            |
| Hard   | 5     | 31            |
| Brutal | 6     | 63            |

## Features

- 4 difficulty levels (3–6 disks)
- Move counter
- Reset / play again without a page reload
- Color-coded disks for easy visual distinction

## Tech Stack

- Vanilla JavaScript (ES5)
- HTML5 / CSS3
- jQuery 2.1

## Run Locally

No build step needed — just open `index.html` in any browser.
